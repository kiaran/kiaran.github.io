PowerIK::GroundSettings Struct

struct POWERIK_LIB_API GroundSettings

The settings used control ground alignment. These settings have no effect unless at least 1 effector has IsGrounded on.

Vec3 Normal

Vector that is normal to the ground surface. Default is +Z.

Vec3 StrideDirection

Normalized vector that describes the direction of travel.

float MaxGroundAngle

Maximum angle relative to gravity vector to apply leaning and stride scaling.

bool ScaleStride

If true, stride of IsGrounded effectors will be scaled in Stride Direction.

float StrideScaleForward

Amount to scale stride relative to the forward vector. Range is 0-1. Default 0.5.

float StrideScaleBack

Amount to scale stride relative to the backward vector. Range is 0-1. Default 0.5.

float StrideScaleSide

Amount to scale stride perpendicular to the forward vector. Range is 0-1. Default 0.5.

bool CounterLean

If True, root bone is rotated in opposite direction of ground normal.

float LeanForward

Amount to lean forward (up hill). Range is 0-1. Default 0.5.

float LeanBack

Amount to lean backward (down hill). Range is 0-1. Default 0.5.

float LeanSide

Amount to lean sideways (side hill). Range is 0-1. Default 0.5.
